## SAP HANA & MRP System Analysis & Architecture – Purchasing (Hoa Phat Group)
### Discription: Led business and system analysis for a SAP-based purchasing system for the Purchasing Department at Hoa Phat Group, focusing on optimizing P2P processes, enhancing supplier management, and designing scalable system architecture aligned with business needs.
### Role: IT Business Analyst
####   + Analyzed end-to-end P2P processes (spend, demand, supplier, contract, cost) and identified stakeholder needs using RASCI & Power/Interest frameworks
####   + Defined enterprise architecture (Business, Data, Application, Technology) and designed 3-tier system architecture
####   + Modeled processes using BPMN, Use Case, and designed system workflows for procurement, supplier management, and purchasing operations
####   + Developed data models (ERD, DFD, Star Schema) and defined data governance and constraints
####   + Designed UI prototypes (Figma) for key modules (Requisition, Supplier, Order)
